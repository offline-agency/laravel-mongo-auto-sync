# Laravel Mongo Auto Sync

## Installation

```bash
composer require offlineagency/laravel-mongo-auto-sync
```

## Documentation
You can find the documentation [here](https://docs.offlineagency.com/laravel-mongo-auto-sync/)

## Testing
Run the tests with:
``` bash
composer test
```
